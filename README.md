# Projeto Homeland

O Projeto Homeland é uma página responsiva desenvolvida no bootcamp TripleTen com base em um layout do Figma. O objetivo do projeto é apresentar uma experiência visual adaptável para diferentes tamanhos de tela, utilizando HTML e CSS organizados com a metodologia BEM.

## Funcionalidades

- Layout responsivo para desktop, tablet e mobile
- Organização de estilos por blocos seguindo BEM
- Integração com layout do Figma
- Estrutura de arquivos organizada para manutenção e escalabilidade
- Publicação online com GitHub Pages

## Tecnologias utilizadas

- HTML
- CSS
- Responsive Design
- Figma
- BEM
